## Basic Usage

---

<br/>

1. Clone this repository
2. do thi maven command to setup the application
   ```
   ./mvnw clean package -DskipTests
   docker build -t task-app .
   docker-compose up
   ```
   and now the access endpoint is  `http://localhost:8080`
